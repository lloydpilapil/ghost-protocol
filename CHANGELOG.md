# Changelog

All notable changes to Ghost Protocol will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] — 2026-03-31

### Added
- **8 Named Specialist Agents**
  - Flight — Lead / Architect
  - EECOM — Core Developer
  - FIDO — Quality Owner
  - Procedures — Prompt Engineer
  - PAO — Docs / DevRel
  - CAPCOM — Integration Specialist
  - RETRO — Security
  - Scribe — Session Logger
- **9 Thread Execution Patterns**
  - Base Thread — single scoped task
  - P-Thread — parallel independent agents
  - C-Thread — sequential phases with human checkpoints
  - L-Thread — extended autonomous execution
  - B-Thread — hierarchical sub-agent coordination
  - F-Thread — competitive multi-strategy evaluation
  - Z-Thread — fully autonomous with self-verification
  - Review Thread — post-execution quality review (Core Four)
  - Status Thread — metrics dashboard and optimization
- **Dual Routing System** — every task routed to Agent + Thread Pattern
- **Core Four Quality Framework** — Context, Model, Prompt, Tools evaluation
- **Persistent Agent Memory** — charter.md + history.md per agent
- **Team Ceremonies** — auto-triggered Design Review and Retrospective
- **Shared Decisions System** — decisions.md with inbox workflow
- **GitHub Copilot Integration** — drop-in copilot-instructions.md
- **Claude Code Integration** — installer detects .claude/ and adds slash commands
- **One-Command Installer** — install.sh for any project
- **Charter Template** — for adding custom agents
- **Next.js Landing Page** — built with Next.js 16, React 19, Tailwind 4, TypeScript 6
